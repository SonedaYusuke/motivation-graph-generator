# モチベーショングラフエディタ

あなたの人生のモチベーションの変遷を視覚化するWebアプリケーション

## 🚀 デモ

GitHub Pagesで公開中: [https://SonedaYusuke.github.io/motivation-graph-generator/](https://SonedaYusuke.github.io/motivation-graph-generator/)

## 📖 使い方

1. 年齢と出来事を入力してデータポイントを追加
2. グラフが自動的に更新される
3. PDFとしてダウンロード可能

## 🛠️ 技術スタック

- HTML5 Canvas
- Tailwind CSS
- JavaScript (Vanilla)
